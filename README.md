# Project-1

Create these packages in the src folder in a new Java Eclipse Project:
    - 'project1'
    - 'project1.managers'
    
Then add the .java files to their specific packages

```
File Tree Setup:
    -src
        -project1
            -project1.manager
                -CreditCardManager.java
                -CustomerManager.java
                -MovieManager.java
                -StarManager.java
            -DBManager.java
            -Proj1.java
```
